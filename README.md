# iot_project
Разработка прототипа ассистивной IoT-системы в виде очков для управления бытовыми устройствами посредством детекции двойного моргания (курсовая работа)

1. **Внутренняя камера (ESP32-CAM + OV3660)**  
   → Постоянный видеопоток  
   → Детекция двойного моргания (2 моргания за 2 секунды)  
   → Отправка триггера на сервер

2. **Сервер (MacBook Air M1)**  
   → Получает событие  
   → Запрашивает снимок с внешней камеры  
   → Распознаёт бытовой прибор через OpenCV  
   → Публикует MQTT-команду `ON` в топик `home/microwave`

3. **Внешняя камера (ESP32-CAM + OV3660)**  
   → Делает снимок по запросу сервера

4. **Умная розетка (ESP8266 + реле SRD-05VDC-SL-C)**  
   → Получает MQTT-команду  
   → Включает бытовой прибор

5. **Питание**  
   → Аккумулятор Li-Po 103450 (2000 мАч, 3.7 В)
